# Privacy Policy - Download Status Sound

**Effective Date:** June 21, 2026

## 1. データ収集について

この拡張機能「Download Status Sound」は、ユーザーの個人情報を一切収集しません。すべての処理はローカル環境（お客様のブラウザ内）で行われ、外部サーバーへのデータ送信は一切行われません。

## 2. 使用されるブラウザAPIとその目的

以下のブラウザAPIを使用しますが、すべてローカル処理であり、取得したデータが外部に送信されることはありません：

| API | 目的 |
|-----|------|
| `chrome.downloads` | ダウンロード状態の監視（完了、エラー、キャンセルなどの検出） |
| `chrome.notifications` | ブラウザネイティブ通知の表示 |
| `chrome.storage` | 拡張機能の設定情報をローカルに保存 |
| `chrome.alarms` | 静かな時間帯などのスケジュール設定 |

## 3. ストレージについて

設定情報（音声ファイルの設定、フィルター設定、通知設定、静かな時間帯の設定など）は、ブラウザのローカルストレージ（`chrome.storage.local`）に保存されます。このデータはお客様のコンピュータ内にのみ保持され、一切外部には送信されません。

### ストレージされる情報の例:
- 各イベント別の音声設定（有効/無効、音量）
- カスタム音声ファイル（データURI形式）
- フィルター設定（除外ドメイン、URLパターン、ファイル拡張子、最小ファイルサイズ）
- 通知設定（メッセージテンプレート）
- 静かな時間帯の設定
- ダウンロードイベント履歴（最大50件）

## 4. ダウンロード履歴について

ダウンロードイベントの履歴がローカルに記録されます。履歴に含まれる情報：

- ファイル名
- ダウンロード状態（完了、エラー、キャンセルなど）
- タイムスタンプ
- ファイルサイズ
- ドメイン情報

これらは拡張機能の動作確認とユーザーの参照目的でのみ使用され、外部には送信されません。

## 5. サードパーティサービス

この拡張機能は以下の点でサードパーティサービスとの連携はありません：

- サードパーティのサーバーやクラウドサービスにデータを送信しません
- アナリティクスやトラッキングコードを含んでいません
- インターネット接続は必要ありません
- アドオンストアからの更新以外では外部と通信しません

## 6. ユーザー設定ファイルについて

ユーザーが設定するカスタム音声ファイル（WAV、OGG、MP3形式）は、ローカルのストレージにデータURI形式で保存されます。これらのファイルは拡張機能の動作中のみメモリ上に読み込まれ、外部に送信されることはありません。

## 7. アクセス権限の説明

本拡張機能は以下のアクセス権限をリクエストします。これらはすべて拡張機能の必須動作に必要なもののみです：

- **downloads**: ダウンロードの完了、エラー、キャンセルなどを監視するため
- **notifications**: ブラウザ通知を表示するため
- **storage**: 設定情報を永続化するため
- **alarms**: 静かな時間帯などの時間ベースの機能を動作させるため

## 8. プライバシーポリシーの変更について

本ポリシーに変更がある場合は、GitHubリポジトリ（[https://github.com/g4biz555-source/projects-public.git](https://github.com/g4biz555-source/projects-public.git)）で公開します。重要な変更がある場合は、拡張機能の更新ノートを通じてお知らせします。

## 9. お問い合わせ

本拡張機能のプライバシーポリシーに関するご質問がある場合は、GitHubリポジトリのIssuesまたはPull Requestにてお問い合わせください。
