# Changelog

## [0.2.0] - 2026-06-11
### Added
- Added the ability to select and register parent domains from a dropdown in the popup.
- Added guard clauses to prevent crashes when processing IPv4/IPv6 addresses or localhost.

### Changed
- Changed the domain display in the popup from plain text to a dropdown menu.
- Adjusted CSS to prevent layout breaks when displaying extremely long domain names.
