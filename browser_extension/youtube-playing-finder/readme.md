# YouTube Playing Finder

YouTube または YouTube Music で動画・音楽を再生中のタブを特定し、アイコンクリックでそのタブへ瞬時にジャンプできるブラウザ拡張機能です。

## 🚀 主な機能

- **再生中タブのカウント表示**: 拡張機能アイコンのバッジに、現在再生中のタブの数が表示されます。
- **クイックジャンプ**: アイコンをクリックすると、再生中のタブへ移動します。
- **複数タブ巡回**: 複数のタブで再生している場合、クリックするたびに順番にタブを切り替えます。
- **タブタイトルの変化**: 再生中のタブのタイトル先頭に `▶` が自動付与され、視覚的に判別しやすくなります。
- **クロスブラウザ対応**: Google Chrome (Manifest V3) および Firefox で動作します。

## PLIVACY POLICY

- 本拡張機能（Youtube Playing Finder）は、YouTubeタブの再生状態を監視し、該当タブへの移動機能を提供するためにのみ「タブ権限」および「YouTubeへのアクセス権限」を使用します。
- ユーザーのブラウジング履歴、個人情報、および利用データを収集・保存することは一切なく、外部サーバーへのデータ送信も行いません。すべての処理はユーザーのブラウザ内で完結します。

## 🛠 インストール方法

現在、ストア（Chrome Web Store / Firefox Add-ons）には公開されていないため、開発者モードでインストールする必要があります。

### Google Chrome / Edge の場合
1. このリポジトリをダウンロード（または `git clone`）します。
2. Chrome を開き、アドレスバーに `chrome://extensions/` を入力して移動します。
3. 右上の「**デベロッパー モード**」をオンにします。
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリックし、このプロジェクトのフォルダを選択します。

### Firefox の場合
1. Firefox を開き、アドレスバーに `about:debugging#/runtime/this-firefox` を入力して移動します。
2. 「**一時的なアドオンを読み込む...**」をクリックします。
3. プロジェクトフォルダ内の `manifest.json` を選択します。

## 📂 ファイル構成

- `manifest.json`: 拡張機能の設定ファイル。
- `background.js`: 再生状態の管理、バッジ更新、タブジャンプ処理を担当。
- `content.js`: YouTube ページ内で再生/停止状態を監視し、タブタイトルを更新。
- `icons/icon.png`: 拡張機能のアイコン画像。

## 📝 使い方

1. YouTube または YouTube Music で動画を再生します。
2. ブラウザのツールバーにあるアイコンに、再生中のタブ数が表示されます。
3. アイコンをクリックすると、そのタブにフォーカスが当たります。
4. 動画を停止すると、バッジのカウントが減り、タイトルの `▶` が消えます。

## ⚠️ 注意事項

- コンテンツスクリプトの仕様上、拡張機能をインストールした直後に既に開いていた YouTube タブは、一度ページをリロード（再読み込み）する必要があります。
- 広告再生中も「再生中」としてカウントされる場合があります。
